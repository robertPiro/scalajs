Landing Page

There are two pages you can go to in this miniscule project:
  • A page with a tiny JS application printing on the console
  • A testing page, making sure one can retrieve content in the /webapp dir.

To run the JS applicaton, one just calls compileAll on the sbt command line. This should, in all theory, place the output files, scala-js-tutorial-fastopt.js and scala-js-tutorial-fastopt.js.map in the /assets/js folder whence it can be retrieved by the first page with the tiny JS application. Admittedly, the JS page looks a bit worse for wear but it does the job as advertised.

For the second page, nothing needs to be done.
